mp3-zing-pydownloader
=====================

### Req.

Install those libraries via `pip`
- urllib3
- progressbar

### Usage

- Download this script
- Run this script with option(s)

``` python
python /path/to/zing.py [OPTION]... [FILE]
  -single <url>
      single song url

  -album <url>  
      album url

  -o
      output directory

```
